Deploy Recall on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy Recall](https://northflank.com/stacks/deploy-recall) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy Recall now**.
4. Set the required environment variable: `RECALL_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the Recall template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the Recall service.
8. Open the public Recall URL at `/recall` and connect using your `RECALL_GATEWAY_TOKEN`.

## What you get

- Hosted Recall Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so config/credentials/workspace survive redeploys

## Connect a channel

Use the Control UI at `/recall` or run `recall onboard` via SSH for channel setup instructions:

- [Telegram](/channels/telegram) (fastest — just a bot token)
- [Discord](/channels/discord)
- [All channels](/channels)

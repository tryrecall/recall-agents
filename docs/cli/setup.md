# `recall setup`

Initialize `~/.recall/recall.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
recall setup
recall setup --workspace ~/.recall/workspace
```

To run onboarding via setup:

```bash
recall setup --wizard
```
